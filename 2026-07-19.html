CLAUDE TIP OF THE DAY — PARALLEL SUBAGENTS

In Claude Code, you can fire multiple independent Agent tool calls in a single response — they run concurrently, not sequentially. Instead of:
1. Check competitor prices → wait → done
2. Scrape reviews → wait → done
3. Audit your listing copy → wait → done

Write one message that spawns all three at once and synthesizes the results afterward. For RABBIO ecom intelligence runs (price check + review scrape + SEO audit), this is 3–5× faster. Add a 4th synthesis agent that merges findings into an action checklist.

Practical: In your next Claude Code session, open a task and in the first message call Agent three times in parallel with separate prompts. Watch them run simultaneously. The context window for each stays clean; you synthesize in the main thread.
