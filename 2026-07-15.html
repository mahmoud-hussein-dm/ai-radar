CLAUDE CODE TIP OF THE DAY — CLAUDE.md Context Injection

Add a CLAUDE.md file to every project root. Claude Code reads it automatically at
session start — you stop re-explaining your stack, brand, and goals every conversation.

Include: store/brand name, tech stack, directories NOT to touch, naming conventions,
recurring constraints, and your current quarter's focus. Claude factors it into every
response without being asked. Solo founders: this is your highest-leverage 5-minute setup.

Example CLAUDE.md for a Shopify project:

# RABBIO Store Context
- Shopify Hydrogen · TypeScript · Klaviyo MCP connected
- Never edit /theme/assets/critical.css
- Product naming: [Material] [Product] [Color] (e.g., "Ribbed Tee Dusty Rose")
- Q3 focus: +20% AOV, improve PDP conversion rate
- Omniuss clients: always wrap AI calls in error handling
- Notion StoryBank: 14dced77-114c-8041a471d31458859ead

Tip: Put your current sprint goal in there too. "Q3 goal: 20% AOV lift" means every
code suggestion Claude makes will be orientated toward that outcome automatically.
