💡 CLAUDE CODE TIP OF THE DAY — Parallel Subagents

Stop running product description rewrites one at a time. Fan out parallel subagents — one per SKU — and a 20-product catalogue job drops from ~20 minutes to ~2 minutes wall-clock. In your CLAUDE.md, add a "## Parallelism rules" section that tells Claude which tasks are safe to run concurrently (no shared file writes) vs. which need to stay sequential. Claude Code respects it automatically and stops bottlenecking on tasks that don't need to be sequential.
