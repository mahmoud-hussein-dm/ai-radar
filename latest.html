CLAUDE CODE TIP OF THE DAY — Verifiable Done-Criteria

End every Claude Code task prompt with a concrete, checkable exit condition.

Instead of: "Build me a Shopify inventory checker"
Write: "Build a Shopify inventory checker that prints a JSON summary to stdout — done when `node check.js` runs without errors and outputs at least one SKU."

Claude knows exactly when to stop. You verify it in one command. No vague "I think it's done" moments, no half-built features. This one habit cuts back-and-forth by ~60%.
