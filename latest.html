1) Anthropic splits Claude Agent SDK billing into its own credit pool
Vendor: Anthropic | Date: June 15, 2026 | Type: feature-drop (billing)

Programmatic/agentic Claude usage — Agent SDK, "claude -p", Claude Code GitHub Actions, third-party Agent SDK apps — no longer draws from your normal Pro/Max chat pool. It now gets a separate monthly $ credit (Pro $20 / Max 5x $100 / Max 20x $200), billed at API rates, requires one-time opt-in, doesn't roll over, and automations just fail once it's gone unless you've enabled overflow billing.
