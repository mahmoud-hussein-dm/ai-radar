⚡ CLAUDE CODE TIP OF THE DAY — VERIFIABLE DONE-CRITERIA

End every non-trivial Claude Code prompt with: "Done when: [specific, observable condition]."

Example: "Build the Shopify webhook handler. Done when: a test POST to /webhook logs the correct order_id to the console without errors."

Claude writes the code AND runs the check before closing the task. Without this, it stops when the code looks right — not when it works. One line saves you multiple back-and-forth turns.
