CLAUDE CODE TIP OF THE DAY — Verifiable Done-Criteria

End every Claude Code task prompt with a testable done-criterion:
"This task is complete when running `npm run build` returns exit code 0 with zero TypeScript errors."

Without it, Claude stops when it THINKS it's done. With it, Claude self-verifies, catches its own mistakes, and only reports success when the bar is actually met. Apply to ecom automations: "Complete when the Klaviyo webhook fires and logs a 200 response in the console."
