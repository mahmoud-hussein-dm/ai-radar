CLAUDE CODE TIP OF THE DAY

Write verifiable done-criteria in CLAUDE.md — especially now that agents run on a schedule.

With today's scheduled deployment launch, your agents execute blind — no human in the loop. Fix: write explicit output contracts in CLAUDE.md. Instead of "summarise today's orders", write: "done = file ./reports/orders-YYYY-MM-DD.json exists, containing keys: total_revenue, order_count, top_sku, anomaly_flags[]". Claude can verify its own output, retry on failure, and you get a real success signal — not just "it ran."
