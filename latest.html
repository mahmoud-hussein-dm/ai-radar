🧠 CLAUDE CODE TIP OF THE DAY

VERIFIABLE DONE-CRITERIA

Add a "## Done When" section to every agentic prompt (or your CLAUDE.md):

## Done When
- All product images have alt-text
- Lint passes with zero warnings
- No console errors in the browser
- Confirmation email sends successfully to test@example.com

Claude will self-check each criterion before declaring the task complete, instead of stopping at "looks good to me." This turns vague instructions into auditable workflows — especially powerful when running Claude Code headlessly on store or agency automations, where you're not watching every step.
