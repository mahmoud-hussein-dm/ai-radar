CLAUDE CODE TIP — PARALLEL SUBAGENTS

Stop running tasks in sequence when they're independent. In Claude Code, spawn multiple agents simultaneously — one audits your Shopify API, another drafts product copy, a third reviews ad scripts — then the orchestrating agent synthesises all three.

Use Agent({run_in_background: true}) for tasks you don't need immediately. The orchestrator continues working and gets notified on completion.
