CLAUDE TIP OF THE DAY — PARALLEL SUBAGENTS

In Claude Code Workflows, wrap independent tasks in parallel([...]) to run them simultaneously. This brief checked 16 vendor sources in one shot — wall-clock time = slowest single fetch, not the sum of all fetches. Practical ecom use: fan out to 5 Shopify API docs pages at once when building a new integration, rather than reading them one by one. Write it as: const results = await parallel(sources.map(s => () => agent(s.prompt)))
