CLAUDE CODE TIP OF THE DAY — VERIFIABLE DONE-CRITERIA

Add a "## Done when" section to every task in your CLAUDE.md or inline task prompt. Claude won't mark it complete until all items are demonstrably checkable — this is the single biggest fix for the "Claude said it was done but it wasn't" problem.

Example:
## Done when
- [ ] Shopify webhook fires in test mode (verify in Shopify event log)
- [ ] Order note appears in Gorgias within 5s
- [ ] No JS errors in browser console

The more specific and checkable your criteria, the more reliably Claude ships working code.
